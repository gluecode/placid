Placid.Response.Hooks

  • Summary
  • Macros

Allows for before and after hooks to a controller. Each hook has the opportunity to modify and/or read the requestâ€™s conn in each stage.

Example

defmodule MyController do
  use Placid.Handler

  before_hook :set_headers
  after_hook :send

  def index(conn, _args) do
    conn |> resp(200, "[]")
  end

  ## Hooks

  def set_headers(conn) do
    conn |> put_resp_header("content-type", "application/json; charset=utf-8")
  end

  def send(conn) do
    conn |> send_resp
  end
end
